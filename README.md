# 🎨 Página "Link-in-Bio" para UI/UX Designer

Este é um projeto de front-end que recria uma página de links no estilo "link-in-bio", ideal para ser usada em perfis de redes sociais. O projeto foi desenvolvido com foco nos princípios de **UI/UX Design**, criando uma interface minimalista, moderna e agradável com **HTML5 e CSS3 puros**.

### **✨ Features e Destaques Técnicos**

* **Layout Moderno com Flexbox**:
    * Toda a interface é perfeitamente centralizada (vertical e horizontalmente) utilizando **CSS Flexbox**. Esta abordagem moderna garante um layout limpo e alinhado com poucas linhas de código.

* **Foco em UI/UX e Design Minimalista**:
    * O projeto foi construído com atenção aos detalhes de UI (User Interface) e UX (User Experience).
    * **Hierarquia Visual**: O uso de diferentes pesos de fonte, espaçamentos e a organização dos elementos criam uma hierarquia clara que guia o olhar do usuário.
    * **Tipografia**: Foram importadas fontes customizadas do Google Fonts para fortalecer a identidade visual do design.

* **Micro-interações com CSS**:
    * Os botões de link possuem um efeito de `hover` sutil, implementado com a propriedade `transition` do CSS, que melhora o feedback visual e a interatividade da página.
    * Sombras (`box-shadow`) foram aplicadas para dar profundidade e destacar os elementos clicáveis.

**Simplesmente clone o repositório e abra o arquivo `index.html` em qualquer navegador web.**
